Section 1

Executive Summary

The Lorain Port & Finance Authority has deployed a custom-built digital platform at lorainport.com that serves as both the public face of the organization and a centralized management hub for all LPFA operations. Unlike off-the-shelf website builders, this platform integrates real-time maritime data, automated event and tour synchronization, vessel traffic monitoring, dual-brand administration (LPFA + Rockin' on the River), and a full content management system — all within a single, unified portal.

Key Differentiator: Dual-Brand Hub

This platform uniquely manages two distinct organizations from a single admin portal. The LPFA side handles port operations, development, events, and public services. The Rockin' on the River side provides a complete event management dashboard with Wix-integrated ticket sales, customer data, inbox, and financial analytics — all accessible to authorized administrators without leaving the platform.

Section 2

Platform Architecture

The platform is built on a modern, enterprise-grade technology stack optimized for performance, security, and scalability.

Frontend Framework

Next.js 15 (App Router) with TypeScript. Server-side rendering, static generation, and incremental revalidation for optimal performance and SEO.

Database & Auth

Supabase — PostgreSQL database with Row-Level Security (RLS), JWT authentication, role-based admin access (super_admin, admin).

Hosting & Deployment

Vercel — Global CDN, automatic Git deployments, serverless functions, cron job scheduling, and zero-downtime deploys.

Design System

Custom CSS with variables, responsive breakpoints, dark mode support, scroll animations, and consistent typography (Montserrat + Inter).

Section 4

Admin Portal — LPFA Management Hub

A full-featured content management system with role-based access, real-time dashboards, and direct database management across 11 modules.

Dashboard Overview

Central hub displaying live counts across all content types: News Articles, Events, Boat Tours, Meeting Minutes, Photos, Staff, Board Members, Vessel Traffic, Active RFPs, Inbox messages, and ROTR upcoming shows — all updated in real-time.

📰 News Management

Create, edit, publish, and schedule news articles. Rich text editor, image uploads, gallery support, category tagging, and SEO-friendly slugs.

📅 Event Management

Full CRUD with year-based grouping. Auto-sync from Wix. Supports headliner/opener fields, gallery images, featured flags, and ticket URLs.

🚢 Boat Tour Management

Tour catalog with monthly schedule management. Auto-syncs availability from PeekPro (OCTO API). Pricing, duration, booking info, and departure details.

👥 Staff & Board Management

Separate modules for team members and board of directors. Photo uploads, bios, contact info, custom sort ordering, and publish/unpublish controls.

📄 Meeting Minutes

Document management for board materials: agendas, minutes, resolutions, and board packets. PDF upload with date filtering and public display.

📋 RFPs & Bids

Procurement management with lifecycle tracking: New → Open → Closed → Archived. Rich text descriptions, document upload, deadline tracking. Public display with collapsible detail cards.

🖼 Photo Library

Centralized image management with tagging. Photo picker component for inserting images into articles, events, and tours.

⚓ Vessel Traffic Monitor

Real-time AIS vessel tracking for Lorain harbor. Displays vessel name, MMSI, destination, position, speed, ETA, and time in port. Status: En Route, In Port, Departed, Expired.

📨 Inbox / Contact Leads

All contact form submissions with status workflow: New → Read → Replied → Archived. Subject categorization, newsletter subscription tracking, and bulk actions.

👤 User Management

Admin user provisioning with invitation system. Role-based access: Super Admin (full control) and Admin (content management). Secure password reset flow.

🔔 Unread Badges

Real-time unread message counts displayed on sidebar menu items for both LPFA Inbox and ROTR Inbox. Auto-refreshes to keep admins informed.

A dedicated management dashboard for the Rockin' on the River concert series, fully integrated with Wix for event, ticket, and customer management.

The ROTR admin panel operates within the same portal as LPFA management. Authorized staff can seamlessly switch between managing port authority operations and concert series administration. This eliminates the need for separate websites, logins, or management tools — reducing cost and complexity.

Summary metrics: total events, upcoming shows, total orders, revenue, and tickets sold. Quick-glance health check of the concert series.

All ROTR events synced from Wix. View event details, dates, venues, ticket tiers, and pricing. Upcoming vs. past event filtering.

Per-event order details: buyer name, email, ticket quantity, order total, confirmation status. Filterable by event.

Customer database from ticket purchases. Contact information, order history, and engagement tracking.

Revenue analytics with date range selection, period comparisons, and trend analysis powered by Wix Analytics API.

Section 7

Automated Systems & Background Jobs

The platform operates several automated processes that run without human intervention, keeping data fresh and reducing administrative overhead.

Vessel Traffic Sync Cron

Connects to AISSTREAM via WebSocket, collects AIS data for 45 seconds, upserts vessel positions to database, and marks stale vessels (>48h unseen) as expired. Automated and hands-off.

Boat Tour Availability Sync Daily 6AM

Pulls product catalog and 90-day availability from PeekPro's OCTO API. Creates or updates tours in database with pricing and monthly schedules. New tours flagged for admin review.

Marine Data Refresh ISR 30min

Aggregates data from NWS (alerts, forecast, hourly, zones), NDBC buoy, and USGS water quality. Cached via Incremental Static Regeneration for fast page loads with fresh data.

Page Cache Revalidation On Save

When admins save content changes, the system automatically revalidates affected public pages via the revalidation API, ensuring visitors see updated content within seconds.

Section 9

Platform Sustainability

Scalability

Built on serverless architecture (Vercel + Supabase), the platform scales automatically with traffic. No server management required. Handles traffic spikes during events like July 4th without manual intervention.

Security

Row-Level Security (RLS) on all database tables. JWT-based authentication. Admin role verification on every protected route. HTTPS everywhere via Vercel. No exposed credentials in client code.

Performance

Global CDN delivery via Vercel. Static pages load in under 1 second. Server-rendered pages with ISR caching. Optimized images and code splitting for fast mobile performance.

Maintainability

TypeScript provides type safety and catches errors before deployment. Single CSS file with design system variables ensures visual consistency. Git version control tracks every change.

Content Independence

Staff can update news, events, tours, staff bios, board info, meeting minutes, RFPs, and photos without developer assistance. The admin portal provides full self-service content management.

Vendor Independence

All code is owned by LPFA. No vendor lock-in. The codebase can be maintained by any qualified Next.js developer. Open-source technologies throughout.
